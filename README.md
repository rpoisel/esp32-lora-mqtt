LoRa to MQTT gateway based on the ESP32 platform.

# Supported Boards

* WiFi LoRa 32 (V1)
* WiFi LoRa 32 (V2)

# Building

1. Install [PlatformIO](https://platformio.org).
1. Build the code using the `platformio run` command.
